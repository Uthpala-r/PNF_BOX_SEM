pub fn save_config(config: &CliConfig) -> Result<()>
Expand description

Saves the given CliConfig to a file named startup-config.json.

This function serializes the provided configuration into JSON format and writes it to a file. If the file already exists, it will be overwritten. If the file does not exist, it will be created. The JSON is formatted for readability (pretty-printed).

  • config: The CliConfig object that contains the configuration to be saved.

This function returns a Result<(), std::io::Error>. It will return Ok(()) if the file is successfully written, or an error if something goes wrong (e.g., file write failure).
