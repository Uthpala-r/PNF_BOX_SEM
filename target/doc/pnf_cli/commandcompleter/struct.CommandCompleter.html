Expand description

A custom completer for the CLI application.

The CommandCompleter provides suggestions for commands based on user input. It integrates with the rustyline crate to offer real-time command-line assistance.

ยงFields

  • commands: A vector of strings containing the list of available commands.
  • current_mode: Gets the current mode of the cli

  • commands: A HashMap where the keys are command names (as String) and the values are vectors of possible completions or subcommands associated with each command.
  • current_mode: The current CLI mode (Mode), which influences the available commands and their completion behavior.

ยงExample
