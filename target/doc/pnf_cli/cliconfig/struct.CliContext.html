Expand description

Represents the current context of the CLI application.

The CliContext maintains the state of the CLI, including the current operational mode, the system prompt, the configuration, and the currently selected interface (if any).

§Examples

let context = CliContext::default();
assert_eq!(context.prompt, "Router>");
